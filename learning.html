JavaScript

// Alert
alert("Hello, World!");

// Console Output
console.log("This is a console log message.");

// DOM Manipulation
document.getElementById("myElement").innerHTML = "This text was changed by JavaScript.";

// Event Handling
function handleClick() {
alert("Button clicked!");
}

Console

1. This is a console log message.
Original text
  1. Alert: The alert() function displays a popup dialog box with the message "Hello, World!".
  2. Console Output: The console.log() function logs the message "This is a console log message." to the browser console. You can view the console output by opening the browser's developer tools.
  3. DOM Manipulation: The document.getElementById() method retrieves an element with the specified ID ("myElement") from the HTML document, and the .innerHTML property is used to modify its content. In this case, the text inside the
    element with the ID "myElement" is changed to "This text was changed by JavaScript."
  4. Event Handling: The handleClick() function is defined, which is triggered when the button is clicked. It displays an alert message saying "Button clicked!".
  5. This example showcases different JavaScript features within an HTML document, including alerts, console output, DOM manipulation, and event handling. It demonstrates how JavaScript code can be embedded and executed within an HTML document to enhance interactivity and functionality.
